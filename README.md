# Blinky-Chaser
Project made through hack club. Many different color LED lights will be on a PCB board and light up. I am designing a custom board that will be a specific shape and in this, learn how to solder and use components. This is my first project that I am doing in Stasis and I am using it to learn how to use Kicad and all of the components necesary to make bigger projects in the future.
